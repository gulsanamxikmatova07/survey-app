# Academic Planning and Mental Clarity Survey

This project is a web application created using Streamlit.

The app allows students to answer questions related to academic planning, productivity, and mental clarity.

Based on the answers, the system calculates a total score and determines the psychological state.

## Features

- student information form
- multiple choice survey questions
- automatic score calculation
- psychological state result
- option to download results as JSON file

## Technologies used

- Python
- Streamlit
- JSON
- GitHub

## How to run locally

Install requirements:

pip install -r requirements.txt

Run the app:

streamlit run app.py
